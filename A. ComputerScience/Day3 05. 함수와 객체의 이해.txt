Day3 05. 함수와 객체의 이해

1. 함수의 이해

	- '재사용' 가능한 명령어
		f(x) = 2x +1
		
		def 함수이름 (파라미터):
		[ 함수에 속한 명령어 ]
		return 리턴값(혹은 식)
		
		ex. f라는 이름의 함수를 정의하고 이를 2회 재사용한다.
		
		 	def f(x):
				return 2*x+1
			y = f(1)
			print(y)
		
			z = f(5)
			print(z)
		
					jupyter는 블록이 다르더라도 
					이전블록을 기억하고 있기 때문에
					재사용가능.

			이 때, print도 함수다. 단, 정의하지 않아도 되는 '내장함수'
		
		내장함수 설명	
		https://docs.python.org/ko/3/reference/index.html 

	- type() 내장함수
		- class 종류 print로 출력할 수 있음.
	- input() 내장함수
		- 화면 상단에 입력상자가 표시됨.
		- 입력한 모든 내용은 (숫자든 문자든) 문자열로 취급.

			* 문자열의 합은 붙여서 출력. a + b = ab.
			print("life" "is" "short"

			* 띄어서 쓰고 싶으면 ,을 사이에 넣을 것.
			print("Life","is","short")
			
			* end는 뭐로 끝나는지 정의하는 것 
			print ("Life", end="~~~")
			* 
		- 입력 내용을 print로 출력할 수 있음
	- 문자열의 정수/실수 변환
		- int(a)
		- float(a)
	- 정수의 실수 변환
		- 소수점 표시
		- a = 1
		- float(a) 	-> 1.0
		- print(a)
	- 실수의 정수 변환
		- 소수점 이하 버림. (반올림 아님)

	- 간단한 연산 기능 함수
		- 제곱근 구하기
			a = pow(2,8)
