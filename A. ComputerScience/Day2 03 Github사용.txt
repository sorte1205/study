	내 컴퓨터에서 진행중인 작업물을 저장소에 등록
		작업 폴더 위치에서 명령프롬프트를 연다 
				(현재 폴더 주소창에 cmd를 치면 됨)
				or (cd /d E:\산대특\본과정 1분기\A. ComputerScience)
		$ git init
		git branch -M main
		git add -A
		git commit -m "간단한 코멘트"
		git remote add origin git@github.com:sorte1205/ComputerScience.git	
					(repository주소, SSH code인지 확인 할 것!)
					주소를 잘못 쓴 경우, .git folder의 config에서 url 수정.
		git push --force --set-upstream origin main
					github이 origin이고, 원본이 된다.
